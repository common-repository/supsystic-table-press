== Installation ==

= To install a plugin via FTP, follow next popup instructions =

1. Download the Table Press by Supsystic plugin
2. Unarchive the Table Press by Supsystic plugin
3. Copy the folder with Table Press by Supsystic plugin
4. Open ftp \wp-content\plugins\
5. Paste the plug-ins folder in the folder
6. Go to admin panel => open item "Plugins" => activate the Table Press by Supsystic plugin
